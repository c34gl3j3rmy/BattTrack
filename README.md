## Fonctionnalités

- 🔋 Suivi de batteries rarement utilisées
- 📈 Estimation d'autodécharge
- 🟢🟠🔴 Statuts visuels
- 💾 Export / import JSON
- 📱 PWA installable sur smartphone
- 🔄 Mise à jour avec affichage des nouveautés
- 🛡️ Notifications locales simples
